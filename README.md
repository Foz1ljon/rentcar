## Rent-Car Application

## launch the project

Create an `.env` file and write variables in it according to the template in the `example.env` file
if not `pnpm`, set it

## Project setup

```bash
 pnpm install
```

## Compile and run the project

```bash
# development
 pnpm run start

# watch mode
 pnpm run start:dev

# production mode
 pnpm run start:prod
```
